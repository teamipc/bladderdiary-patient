# Plan 06-10: `DayPageClient.tsx` orchestration — wire `onDirtyChange` from all 5 forms, intercept BottomSheet close to show ConfirmDialog when dirty, pass `maxWidth='narrow'` for bedtime/wakeup forms, pass `inert={pendingClose}` to suspend BottomSheet focus trap when ConfirmDialog stacks

<objective>
Per 06-UI-SPEC §"Reset-on-Cancel Pattern" → "Implementation pattern (DayPageClient owns the ConfirmDialog)" (locked), DayPageClient is the orchestration layer that:

1. **Tracks dirty state of whichever form is currently open** via `const [activeFormDirty, setActiveFormDirty] = useState(false);` and passes `onDirtyChange={setActiveFormDirty}` to each of the 5 form components rendered inside BottomSheet
2. **Intercepts BottomSheet `onClose`** — if `activeFormDirty`, show ConfirmDialog instead of immediately closing; if NOT dirty, close silently (existing behavior preserved)
3. **Renders ConfirmDialog** with 4 i18n keys from Plan 06-02 (`tc('discardEntryTitle')` etc.), `variant="danger"`
4. **Passes `inert={pendingClose}`** to BottomSheet so the BottomSheet's focus trap (Plan 06-04) is suspended while ConfirmDialog is stacked above it, avoiding the stacked-modal focus-trap conflict noted in 06-UI-SPEC §"Reset-on-Cancel Pattern" → "ConfirmDialog wiring" → "Stacked-modal behavior"
5. **Passes `maxWidth='narrow'`** to BottomSheet when `sheetMode === 'bedtime' || sheetMode === 'wakeup'` — these slider-only forms use the `max-w-2xl` modal at md+ per 06-UI-SPEC §"Summary Table" → SetBedtime/SetWake rows
6. **Resets `activeFormDirty` to false on every form open/close transition** to avoid stale dirty-state leaking across form-mode switches

This plan is the FINAL wiring step before verification (Plan 06-11). It depends on EVERY prior wave that produces consumed contracts:
- 06-02 (i18n keys must exist for `tc('discardEntryTitle')` etc.)
- 06-04 (BottomSheet `maxWidth` + `inert` props must exist)

Runs alone in Wave 5 (no parallel work).

    ```ts
    // Used by handleSave (after a successful save) AND by handleDiscardConfirm

    const closeSheet = useCallback(() => {
      setSheetMode(null);
      setEditVoidEntry(undefined);

      setEditLeakEntry(undefined);
      setInitialTime(undefined);
      setActiveFormDirty(false);  // Reset dirty-state on every close transition.

    // Public close gate — fired by BottomSheet's onClose (close X / Escape / backdrop click).
    // If the active form is dirty, show ConfirmDialog; otherwise close silently.
    const handleSheetClose = useCallback(() => {
      if (activeFormDirty) {
        setPendingClose(true);
      } else {
        closeSheet();
      }
    }, [activeFormDirty, closeSheet]);

    const handleDiscardConfirm = useCallback(() => {
      setPendingClose(false);
      closeSheet();
    }, [closeSheet]);

    const handleDiscardCancel = useCallback(() => {
      setPendingClose(false);
      // Keep modal open; user returns to editing.
    }, []);
    ```
