# Plan 05-02: Add `nav.primaryNavAriaLabel` i18n key (English source + 5-locale mirrors)

<objective>

This plan is Wave 1, blocks 05-04 (Header). It has no `depends_on` and runs in parallel with 05-01.

- Collaborative tone ‚Äî "Primary navigation" is neutral landmark label, not authoritative copy
