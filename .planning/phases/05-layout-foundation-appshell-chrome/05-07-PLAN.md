# Plan 05-07: Multi-width / multi-locale Phase 5 chrome verification (local build + screenshots + automated invariant checks)

<objective>
Verify that all six Phase 5 success criteria from ROADMAP §"Phase 5: Layout foundation + AppShell chrome" hold against the actual built site, across the 6-locale × {375px / 768px / 1280px} viewport matrix specified in UI-SPEC §"Verification Checklist (for Phase 8 visual-qa pass)".

1. **Verifies six DTUX-02 success criteria via DOM assertions and screenshots:**
